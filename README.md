# AI Stock Prediction Application

## Description

AI Stock Prediction is a web application that analyzes stock market data and generates price predictions using artificial intelligence. The application allows users to view stock information, analyze trends, and receive forecasted market insights. It uses Google Gemini.

## Technologies Used

* Java
* TypeScript
* HTML
* JSON

## Features

* Stock price prediction
* Historical data analysis
* Real-time stock information
* Interactive charts
* User-friendly web interface

## Disclaimer

This application is intended for educational purposes only. Stock predictions are estimates and should not be used as financial advice.
